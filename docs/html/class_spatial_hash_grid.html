Detailed Description

Uniform spatial hash grid for broad-phase collision candidate generation.

Partitions world space into square cells of side cellSize (in world units). Objects with colliders are inserted into one or more cells based on their world position and Collider::GetBoundingRadius. Objects that are too large relative to cellSize are tracked in largeObjects.

Pipeline:

  • Clear resets internal buckets for the current frame.
  • Insert assigns each object to all cells touched by its bounding circle, or to largeObjects if it spans many cells.
  • ComputeCollisions iterates buckets and invokes the supplied callback for each unique candidate pair (Object*, Object*). Mask/group filtering is performed at a higher layer (ObjectManager).
Warning
The grid is rebuilt every frame; choose cellSize carefully to balance insertion cost and bucket density.

Detailed Description

A simple 2D camera for managing position, zoom, and screen size.

Camera2D provides functionality for transforming world coordinates into screen-space via view matrices. It supports zoom, screen resizing, and viewport culling checks. Typically used by RenderManager and CameraManager to control scene rendering.

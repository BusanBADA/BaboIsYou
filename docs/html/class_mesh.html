Detailed Description

GPU-backed draw-able geometry (VAO/VBO/EBO) with optional instancing.

Mesh uploads vertex/index data to GPU buffers and exposes non-instanced and instanced draw paths. It also computes a conservative local half-size used for culling or bounds estimates.

The instancing path expects per-instance: model matrix, color, UV offset and UV scale.

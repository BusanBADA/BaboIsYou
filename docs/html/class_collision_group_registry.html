Detailed Description

Registry mapping collision group tags to bitmasks.

Provides stable bit assignments for textual group tags so that Objects can configure collisionCategory and collisionMask. The first unseen tag is assigned the next available bit index.

Bidirectional maps:

  • tagToBit : std::string -> bit index
  • bitToTag : bit index -> std::string
Note
Bits are assigned monotonically via currentBit; ensure consumers handle cases where the bit width limit could be reached.

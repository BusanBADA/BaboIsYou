Detailed Description

Default implementation of a basic in-game object.

GameObject is a concrete subclass of Object with type = GAME. It provides empty overrides for all lifecycle methods (Init, Update, Draw, Free, etc.), making it suitable as a lightweight placeholder or a base for user-defined objects.

Typical usage:

  • Inherit from GameObject to create your own gameplay objects.
  • Override only the lifecycle functions you need.
class Player : public GameObject {
public:
void Init(const EngineContext& ctx) override {
SetTag("[Object]Player");
SetMaterial(ctx, "[Mat]player");
}
void Update(float dt, const EngineContext& ctx) override {
// Player-specific logic
}
};

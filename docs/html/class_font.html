Detailed Description

Font class for text rendering using FreeType.

Font handles glyph extraction from a TTF file, dynamic atlas baking, and text mesh generation. Each Font maintains its own texture atlas and associated Material used for rendering text.

Atlas baking uses a "lazy baking" strategy: missing glyphs are added to the atlas when requested. The atlas can expand if more space is needed.

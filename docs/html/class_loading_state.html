Detailed Description

Transitional game state that asynchronously loads resources before entering the next state.

LoadingState owns an AsyncResourceLoader and exposes queue functions to register textures, shaders, fonts, and sounds. When all jobs are done it constructs the next GameState using nextFactory and hands off control to it.

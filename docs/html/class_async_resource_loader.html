Detailed Description

Background worker that asynchronously loads engine resources (textures, shaders, fonts, sounds).

AsyncResourceLoader runs a background thread that processes queued LoadRequest jobs. Each completed load produces a LoadResult which can be fetched and uploaded to GPU or audio engine.

Typical flow:

  • Call Start to launch the worker thread.
  • Enqueue resource jobs with Enqueue.
  • Periodically call FetchCompleted to retrieve finished results and register them with engine managers.
  • Call Stop when shutting down or leaving the loading state.

Thread safety:

  • Public API is thread-safe for enqueueing jobs and fetching results.
  • Internal synchronization uses a mutex + condition variable to wake the worker thread.

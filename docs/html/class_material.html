Detailed Description

High-level wrapper that binds a shader with uniforms and textures.

Material manages a Shader, a set of typed uniforms, and a set of named texture bindings. It also exposes optional GPU instancing toggles if the underlying Shader/Mesh pair supports it.

Typical usage:

Material* mat = rm->GetMaterialByTag("[Mat]sprite");
mat->SetUniform("u_Tint", glm::vec4(1,1,1,1));
mat->SetTexture("u_MainTex", rm->GetTextureByTag("[Tex]player"));

\doxysubsection{Detailed Description}
Represents a single frame of a sprite sheet. 

Each \doxylink{struct_sprite_frame}{Sprite\+Frame} stores UV coordinates for texture sampling, the pixel size of the frame, and an optional offset for alignment.

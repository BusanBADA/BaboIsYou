\doxysubsection{Detailed Description}
Represents a grid-\/based sprite sheet backed by a texture. 

The \doxylink{class_sprite_sheet}{Sprite\+Sheet} divides a texture into frames of fixed width and height, supports querying UV offsets/scales for a given frame, and can store named animation clips referencing frame sequences.

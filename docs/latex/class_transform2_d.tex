\doxysubsection{Detailed Description}
Represents a 2D transform with position, rotation, scale, and depth. 

\doxylink{class_transform2_d}{Transform2D} encapsulates spatial data for objects in 2D space, including position, rotation (in radians), scale, and rendering depth. It maintains a transformation matrix that is lazily updated only when values change, minimizing redundant matrix computations.

Retrieves the transformation matrix. 

If any property (position, rotation, scale, depth) has changed, this function recalculates the matrix as T \texorpdfstring{$\ast$}{*} R \texorpdfstring{$\ast$}{*} S, where\+:
\begin{DoxyItemize}
\item T = translation matrix
\item R = rotation around Z axis
\item S = scale matrix
\end{DoxyItemize}

Otherwise, it returns the cached matrix.

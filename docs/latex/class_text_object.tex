\doxysubsection{Detailed Description}
Scene object specialized for rendering 2D text. 

\doxylink{class_text_object}{Text\+Object} produces a mesh from a UTF-\/8 string using the provided \doxylink{class_font}{Font}\textquotesingle{}s dynamic glyph atlas. It honors horizontal/vertical alignment, integrates with the engine\textquotesingle{}s camera rules (including HUD-\/style "{}ignore camera"{}), and updates its mesh lazily when the text or font atlas changes.

\begin{DoxyNote}{Note}
Material/\+Mesh setters and animator APIs are intentionally deleted to prevent misuse. \doxylink{class_text_object}{Text\+Object} internally owns its mesh and uses the Font-\/provided material. 
\end{DoxyNote}

\doxysubsection{Detailed Description}
High-\/level wrapper that binds a shader with uniforms and textures. 

\doxylink{class_material}{Material} manages a \doxylink{class_shader}{Shader}, a set of typed uniforms, and a set of named texture bindings. It also exposes optional GPU instancing toggles if the underlying Shader/\+Mesh pair supports it.

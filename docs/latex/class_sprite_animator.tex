\doxysubsection{Detailed Description}
Time-\/based animator for playing sprite sheet frames and clips. 

The \doxylink{class_sprite_animator}{Sprite\+Animator} advances frames based on elapsed time and plays either a continuous frame range or a named \doxylink{struct_sprite_clip}{Sprite\+Clip}.

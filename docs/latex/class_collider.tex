\doxysubsection{Detailed Description}
Abstract base for 2D colliders attached to an \doxylink{class_object}{Object}. 

A \doxylink{class_collider}{Collider} defines a 2D collision shape for its owning \doxylink{class_object}{Object} and exposes a uniform API for point tests, pairwise collision tests, and debug drawing.

Integration \& flow\+:
\begin{DoxyItemize}
\item Ownership\+: a \doxylink{class_collider}{Collider} is owned by an \doxylink{class_object}{Object} (see \doxylink{class_collider_ae661027024ee407d042b8672625da377}{owner}). Lifetime is tied to the \doxylink{class_object}{Object}\textquotesingle{}s lifetime.
\item Broad phase\+: \doxylink{class_spatial_hash_grid}{Spatial\+Hash\+Grid} partitions Objects and enumerates candidate pairs; \doxylink{class_object_manager}{Object\+Manager} then calls \doxylink{class_collider_a41660505ae4b413d3d4b004162857426}{Check\+Collision} on those pairs.
\item Narrow phase\+: \doxylink{class_collider_a41660505ae4b413d3d4b004162857426}{Check\+Collision} uses double dispatch via \doxylink{class_collider_a6b1071065ccb7ddfc1ff8340a89246d7}{Dispatch\+Against} to invoke the correct shape–shape test.
\item Transform\+: if \doxylink{class_collider_a10c23d55e50b048078cddb0f806d5731}{use\+Transform\+Scale} is true, \doxylink{class_collider_a188380ff8ece20474e86a662f995887e}{Sync\+With\+Transform\+Scale} is called from \doxylink{class_object_manager}{Object\+Manager} each frame to update internal scaled dimensions.
\item Debug\+: \doxylink{class_collider_a222e52491ae094943420952fd3182c90}{Draw\+Debug} renders the shape using \doxylink{class_render_manager}{Render\+Manager} in world space.
\end{DoxyItemize}

Notes\+:
\begin{DoxyItemize}
\item All non-\/const behavioral APIs are virtual; concrete shapes (circle, AABB) must implement them.
\item Point coordinate spaces\+: point and rendering are in world space; shapes should incorporate the owner\textquotesingle{}s transform and \doxylink{class_collider_a7a84eff1697e0c8d54ef0df6427a8235}{offset} when testing. 
\end{DoxyItemize}

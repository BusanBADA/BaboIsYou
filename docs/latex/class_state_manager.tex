\doxysubsection{Detailed Description}
Manages game states and their transitions. 

The \doxylink{class_state_manager}{State\+Manager} holds the currently active \doxylink{class_game_state}{Game\+State} and handles transitions to new states. It ensures proper unloading, freeing, and reinitialization of resources when switching between states. Internally, the manager defers state changes until the next update tick to avoid inconsistent state transitions.

Example usage in engine loop\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{stateManager.Update(dt,\ engineContext);}
\DoxyCodeLine{stateManager.Draw(engineContext);}

\end{DoxyCode}

Requests a change to a new \doxylink{class_game_state}{Game\+State}. 

The new state will not be activated immediately. Instead, it is stored in {\ttfamily next\+State} and will be applied during the next {\ttfamily \doxylink{class_state_manager_ab4ebd3189a4623e805d95f768b314acb}{Update()}} call. This ensures a clean transition with proper resource unloading and loading.

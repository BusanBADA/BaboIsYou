\doxysubsection{Detailed Description}
Central context holding pointers to all core engine subsystems. 

The \doxylink{struct_engine_context}{Engine\+Context} structure aggregates references to the major subsystems of the engine, providing a unified way for game states, objects, and managers to access engine services.

It is initialized inside {\ttfamily \doxylink{class_s_n_a_k_e___engine_aa6891c4512199870f8ee25cc6d82e45a}{SNAKE\+\_\+\+Engine\+::\+Set\+Engine\+Context()}} and passed throughout the engine to ensure all systems can communicate without needing direct global access.

Members include\+:
\begin{DoxyItemize}
\item \doxylink{class_state_manager}{State\+Manager}\+: Manages game state transitions, updates, and rendering calls.
\item \doxylink{class_window_manager}{Window\+Manager}\+: Handles window creation, resizing, fullscreen toggling, and input callbacks.
\item \doxylink{class_input_manager}{Input\+Manager}\+: Provides input querying (keys, mouse, scroll, world positions).
\item \doxylink{class_render_manager}{Render\+Manager}\+: Responsible for resource registration, frame rendering, and draw batching.
\item \doxylink{class_sound_manager}{Sound\+Manager}\+: Manages audio playback, volume control, and cleanup of sound instances.
\item \doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine}\+: Back-\/reference to the main engine instance, used for global operations such as quitting. 
\end{DoxyItemize}

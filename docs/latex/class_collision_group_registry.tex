\doxysubsection{Detailed Description}
Registry mapping collision group tags to bitmasks. 

Provides stable bit assignments for textual group tags so that Objects can configure {\ttfamily collision\+Category} and {\ttfamily collision\+Mask}. The first unseen tag is assigned the next available bit index.

Bidirectional maps\+:
\begin{DoxyItemize}
\item \doxylink{class_collision_group_registry_ab95c6214265044dd64b94eede30bf228}{tag\+To\+Bit} \+: std\+::string -\/\texorpdfstring{$>$}{>} bit index
\item \doxylink{class_collision_group_registry_a03eaa11789a579244d7352b9bbdf07f1}{bit\+To\+Tag} \+: bit index -\/\texorpdfstring{$>$}{>} std\+::string
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Bits are assigned monotonically via \doxylink{class_collision_group_registry_a56e97b092b3b9a4872dc4ad796d00443}{current\+Bit}; ensure consumers handle cases where the bit width limit could be reached. 
\end{DoxyNote}

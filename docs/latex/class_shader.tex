\doxysubsection{Detailed Description}
Encapsulates an Open\+GL shader program. 

Provides functionality to attach and compile shader sources from files or strings, link them into a program, and set uniform variables. Also tracks whether the program supports GPU instancing based on the presence of specific attributes.

\doxylink{class_shader}{Shader} objects are owned and managed by \doxylink{class_render_manager}{Render\+Manager} and \doxylink{class_material}{Material}. They are typically created during resource loading and bound before rendering objects.

\doxysubsection{Detailed Description}
Core engine class that manages all subsystems and the main loop. 

\doxylink{class_s_n_a_k_e___engine}{SNAKE\+\_\+\+Engine} is the root object that initializes subsystems (window, input, rendering, audio, and states), manages the game loop, and ensures proper shutdown. It exposes a reference to \doxylink{struct_engine_context}{Engine\+Context} for access to subsystems, and provides entry points for initialization, execution, quitting, and debug rendering control.

Runs the main engine loop. 

Executes the loop until \doxylink{class_s_n_a_k_e___engine_abc2c17ba271c262a62bf2f06ec8334d4}{Request\+Quit()} is called. Each frame performs\+:
\begin{DoxyItemize}
\item Event polling
\item Delta time calculation
\item \doxylink{class_state_manager}{State\+Manager} update and draw
\item Debug draw (if enabled)
\item Swap buffers
\end{DoxyItemize}

This function blocks until the engine shuts down.

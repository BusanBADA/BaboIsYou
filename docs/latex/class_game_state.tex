\doxysubsection{Detailed Description}
Abstract base class representing a game state in the engine. 

\doxylink{class_game_state}{Game\+State} defines the lifecycle and update/draw pipeline of a state (e.\+g., title screen, main menu, gameplay). It owns its own \doxylink{class_object_manager}{Object\+Manager} and \doxylink{class_camera_manager}{Camera\+Manager}, and exposes overridable hooks for initialization, updating, drawing, and cleanup.

The engine calls internal System\texorpdfstring{$\ast$}{*} functions to guarantee correct ordering of Init, Update, Collision checks, Draw, and Free steps. Derived classes should override the protected virtual hooks (Init, Update, Draw, etc.) rather than calling System\texorpdfstring{$\ast$}{*} directly.

\doxysubsection{Detailed Description}
Holds text rendering parameters for a text instance. 

A lightweight bundle that pairs a font resource with the UTF-\/8 text string to render. This is consumed by \doxylink{class_text_object}{Text\+Object} to build/update a mesh of quads per-\/glyph using the engine\textquotesingle{}s dynamic glyph atlas.

\doxysubsection{Detailed Description}
Manages sound resources and playback through miniaudio. 

\doxylink{class_sound_manager}{Sound\+Manager} loads audio files, creates playback instances, and provides control over volume, pausing, resuming, and stopping. Each sound instance is identified by a unique \doxylink{_sound_manager_8h_a7675718ce8055fa8280410c0d32ad4f3}{Sound\+Instance\+ID}, which can be reused for efficiency.

Sounds are registered by a string tag and can be controlled individually or in groups (by tag or globally).

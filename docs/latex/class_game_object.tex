\doxysubsection{Detailed Description}
Default implementation of a basic in-\/game object. 

\doxylink{class_game_object}{Game\+Object} is a concrete subclass of \doxylink{class_object}{Object} with type = GAME. It provides empty overrides for all lifecycle methods (Init, Update, Draw, Free, etc.), making it suitable as a lightweight placeholder or a base for user-\/defined objects.

Typical usage\+:
\begin{DoxyItemize}
\item Inherit from \doxylink{class_game_object}{Game\+Object} to create your own gameplay objects.
\item Override only the lifecycle functions you need.
\end{DoxyItemize}

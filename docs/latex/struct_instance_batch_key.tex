\doxysubsection{Detailed Description}
Key structure used for grouping objects into instanced batches. 

The batch key is defined by a unique combination of\+:
\begin{DoxyItemize}
\item \doxylink{class_mesh}{Mesh} pointer
\item \doxylink{class_material}{Material} pointer
\item \doxylink{class_sprite_sheet}{Sprite\+Sheet} pointer
\end{DoxyItemize}

Objects that share the same key can be drawn together using GPU instancing.
